### Install

* `$ git clone git@github.com:MateusGabi/GabUI.git`
* `$ npm install`
* `$ npm run dist`

You can see some examples in `examples` directory.

### Edit output files

* Edit lines 15, 16 and 17 on `Gulpfile.js`
* Edit line 3 on `webpack.config.js`
